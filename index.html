How to Get Your Gemini API Key (Required)

This tool uses the Google Gemini API for both Text-to-Speech (TTS) and Vision (OCR). You need an API key with access to these capabilities.

  1. Go to Google AI Studio: Visit aistudio.google.com.
  2. Sign In: You must be logged into a Google account.
  3. Create API Key: Navigate to the "Get API Key" section (usually found on the sidebar or profile menu) and Create API Key in New Project.
  4. Enable APIs: Ensure that the necessary APIs (like Vertex AI API or Generative Language API, depending on the current platform structure) are enabled for the project associated with your key.
  5. Paste Key: Copy the generated key and paste it into the Custom Gemini API Key field above in the Settings tab.
